# Human Audit Reviewer Packets

These files are documentation-only handoff packets. They copy source audit rows and leave reviewer label fields blank.

Do not cite these files as human-audit evidence. Human-audit numbers can be reported only after `labeled_human_audit_queries.csv`, `human_audit_labeling_manifest.json`, and `human_audit_summary.json` exist and `scripts/18_verify_human_audit_labels.py` reports no failures.
