# SQE Paper Build

This directory contains the generated paper source and compiled PDF.

Build only the PDF:

```bash
/home/nlp-07/.local/bin/tectonic-musl main.tex
```

Regenerate and verify the deterministic paper package from existing executed
results for seeds 42, 43, 44, 45, 46, 47, 48, and 49:

```bash
bash /home/nlp-07/sqe_experiment/scripts/run_paper_pipeline.sh
```

Current verified outputs:

- `main.pdf`: 11-page compiled PDF as of the latest clean build audit.
- `main.tex`: generated LaTeX source.
- `main_conference_preview.pdf`: optional two-column layout preview generated
  from the same source. It is not an official venue template and is not
  experiment evidence.
- `main_conference_preview.tex`: generated source for the two-column preview.
- `tables/`: result, cost, paired-test, gate, case-analysis, and multiseed tables.
- `figures/`: generated method, retrieval, and gate diagnostic figures.

Active tables are the files imported by `main.tex`: `experiment_manifest`,
`main_results`, `multiseed_summary`, `multiseed_paired_tests`,
`win_loss_analysis`, `cost_summary`, `measured_token_cost`, `paired_tests`,
`gate_diagnostics`, `threshold_sweep`, `validation_threshold`,
`multiseed_gate_validation`, `gate_validation_paired_tests`,
`cross_seed_top1_gate`, `gate_variant_diagnostics`,
`gate_headroom_diagnostics`, and `case_analysis`. Other files under `tables/` may be retained
historical/generated artifacts and should not be cited as active paper evidence
unless they are imported by `main.tex` or documented in
`/home/nlp-07/sqe_experiment/DATA_PROVENANCE.md`.

Active figures are the files imported by `main.tex`: `method_overview.png`,
`recall_at_5.png`, `gate_diagnostic.png`, and
`threshold_sensitivity.png`. They are generated by
`/home/nlp-07/sqe_experiment/scripts/06_make_paper_artifacts.py`, numbered in
the compiled PDF, and referenced in the body text.

The machine-readable active/retained table inventory is
`/home/nlp-07/sqe_experiment/paper/table_inventory.json`.
Its `active_table_sources` field maps every table imported by `main.tex` to
the concrete JSON or JSONL files used to generate it. The verifier fails if an
active table has no declared source files or if any declared source file is
missing.

The paper intentionally keeps the GitHub URL as a placeholder until a public
repository is created. It also keeps the Hugging Face dataset URL as a
placeholder until the local release package is uploaded. The local release helper
is `/home/nlp-07/sqe_experiment/scripts/33_prepare_hf_dataset_release.py`.

## Evidence Policy

Only executed result artifacts may be used as paper evidence. In particular,
`Random-Gated-Expansion` refers to the executed random-gating budget baseline
stored in `results_500_memory_seed*/random_budget_detailed.jsonl` and matching
summary files. It is backed by executed JSONL rows.

Deprecated or diagnostic-only files must not be cited as active paper evidence.
Pass@1 schemas, human-audit templates, and unlabeled human-audit packets are
documentation or preparation artifacts only; they are not result evidence.

The current package is a rigorous retrieval-paper draft, not a complete
top-conference empirical submission. See:

- `/home/nlp-07/sqe_experiment/SUBMISSION_READINESS.json`
- `/home/nlp-07/sqe_experiment/ARTIFACT_MANIFEST.json`
- `/home/nlp-07/sqe_experiment/results_500_memory_seed42/verification_report.json`

The active verifier checks that paper tables match result JSON/JSONL files, the
compiled PDF contains the expected paper sections, excluded non-evidence tables
and invalid dataset artifacts are not referenced, and the checksum manifest is
fresh.
