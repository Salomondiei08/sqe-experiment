# SQE Conference Slides

Open `sqe_conference_slides.html` in a browser.

Controls:
- Right arrow, space, or `Next`: advance
- Left arrow, backspace, or `Back`: go back
- `Home`: first slide
- `End`: last slide

Speaker notes are in `sqe_conference_speaker_notes.md`.

The deck uses only verified retrieval results from the local SQE experiment artifacts. It does not claim downstream Pass@1 results or human-audited labels.
